# COVID-19 Vaccination Sentiment Analysis

## Overview
This project was completed as part of my MSc Data Science dissertation at the University of Salford.

The objective was to analyse public sentiment towards COVID-19 vaccination using Natural Language Processing (NLP) and machine learning techniques.

## My Contribution
I independently developed the complete data analysis pipeline including:

- Data collection and preprocessing
- Text cleaning and feature engineering
- Sentiment classification
- Model training and evaluation
- Visualisation and reporting

## Technologies
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Matplotlib

## Results
The project successfully classified vaccination-related sentiment and provided insights into public opinion trends.

## Publication
This work was later published at the International Conference on Data Science, AI and Applications.
